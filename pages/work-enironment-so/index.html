Intro

Objetivo: Conseguir que el desktop environment de Lubuntu (LXQT / OpenBox) cumpla ciertas funcionalidades quasi de un tiling window manager (WM). Lubuntu utiliza como desktop environment LXQT y este a su vez utiliza OpenBox como window manager. No obstante, OpenBox es un floating WM (de esos a los que estamos acostumbrados donde puedes arrastrar las ventanas y redimensionarlas con el ratón colocándolas donde te apetezca), en lugar de un tiling WM (los cuales reubican las ventanas para aprovechar el espacio de la pantalla al máximo y se operan con el teclado en lugar de con el ratón). Si OpenBox es el WM que utiliza Lubuntu, bastará con cambiar la configuración de OpenBox ¿verdad? Pues a grandes rasgos sí, pero habrá que tener en cuenta un par de consideraciones si queremos utilizar algunos atajos de teclado, puesto que Lubuntu y LXQT utilizan unos shortcuts que podrían interferir con nuestra configuración de teclas.

Configuración de OpenBox

Vamos a añadir los atajos de teclado que queremos que nuestro WM (OpenBox) utilice. Para ello tendremos que abrir y editar el archivo de configuración de openbox $HOME/.config/openbox/rc.xml
